# #65 Aspect Ratio Sync

A minimalist utility for creators to synchronize and calculate pixel dimensions across various aspect ratios. It eliminates the need for manual calculations when resizing videos or images for different social media platforms.

## Core Philosophy
* **1 Tool / 1 Action**: Enter one dimension (Width or Height), and instantly get the other based on your chosen ratio.
* **Bi-directional Sync**: Seamlessly calculates Width from Height, or Height from Width, depending on which field you edit.
* **Tap-to-Copy Efficiency**: Built for high-speed workflows—tap any value to copy it to your clipboard and automatically select it for the next edit.

## Key Features
* **Popular Presets**: One-tap access to industry-standard ratios: 16:9 (HD), 9:16 (SNS), 4:5 (Instagram), 1:1 (Square), 4:3 (SD), and 21:9 (Cinema).
* **Custom Ratio Support**: Flexibility to input any non-standard ratio for specialized projects.
* **Instant Clear**: Dedicated 'clr' buttons for each input field to reset and start a new calculation in a single tap.
* **Visual Feedback**: Real-time border highlighting and "Copied!" toast notifications for a responsive, tactile feel.

## Design System
* **Background**: #fff
* **Main Accent**: #333 (for active buttons and focus)
* **Input Style**: 15px Rounded Corners with iOS-optimized typography.
* **Clean UI**: Minimalist grid layout that remains readable and functional on all mobile screen sizes.

## Performance
* **Browser-only**: Logic runs entirely on the client side for instant results.
* **Ultra-Lightweight**: Zero external dependencies, ensuring snappy performance even on low-spec devices.
